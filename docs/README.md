# Documentation Folder

This folder contains documentation for the UVA Intramural Soccer League database project.

## Files

- design.md
  - Describes the overall database design and system design.

- requirements.md
  - Lists functional requirements and project requirements.

- business_rules.md
  - Lists relationship-focused database business rules.

- security_requirements.md
  - Describes user roles, permissions, and privacy restrictions.

- status_report_2_notes.md
  - Tracks feedback updates and remaining work for the second status report.

## Notes

Business rules are intentionally separated from security and application requirements. Business rules focus on relationships, cardinality, and data model constraints, while security requirements focus on user access and privacy.
